# Portfolio — Arne Klimmt

Personal portfolio website for Arne R. Klimmt, Frontend Developer.

## Tech Stack

- Vanilla HTML5, CSS3, JavaScript — no build tools or frameworks
- PHP for contact form mail handling

## Features

- Bilingual (EN / DE) with client-side i18n via `data-i18n` attributes
- Responsive layout with mobile-first breakpoints
- Burger navigation for mobile
- Contact form with client-side validation and server-side PHP mailer
- Portfolio showcase with live demo and GitHub links
- Legal pages (Privacy Policy, Legal Notice)


## Local Development

No build step required — open `index.html` directly in a browser or serve with any static file server.

For the contact form, a PHP-capable server is needed.
